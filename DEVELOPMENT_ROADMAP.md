# Minimail 开发路线图

> 完整开发计划见 [`DEVELOPMENT_PLAN.md`](DEVELOPMENT_PLAN.md)

---

## 已完成

| 阶段 | 内容 | 状态 |
|------|------|------|
| **Phase 0** | 项目基建 — FastAPI 骨架、PostgreSQL、用户认证、Redis、Alembic、Docker、CI、前端基建 (Vite + React + Tailwind + Zustand) | ✅ 完成 |
| **Phase 1** | 核心邮件 — IMAP 连接池、文件夹列表、邮件分页、MIME 解析、三栏布局 | ✅ 完成 |
| **Phase 2** | 设置 — IMAP/SMTP 配置 API、密码加密 (Fernet)、前端配置表单、邮件账户设置面板 | ✅ 完成 |
| **Phase 3** | 发送 — SMTP 发送 API (SSL/STARTTLS)、MIME 构建、前端撰写页面 (收件人自动完成、格式校验) | ✅ 完成 |
| **Phase 4** | 通讯录 — Contact + ContactGroup ORM、CRUD API + 自动完成、前端通讯录面板 (搜索/分组/防抖) | ✅ 完成 |
| **UI 审计** | 侧栏导航、退出登录、无配置引导、API 密钥面板、工具栏头部统一、自适应修复、安全修复 | ✅ 完成 |

## 待开发

| 阶段 | 内容 | 优先级 |
|------|------|--------|
| **收件人自动完成** | 撰写页 To/Cc/Bcc 接入通讯录 autocomplete API | ⏳ 已完成 |
| **附件管理** | 预览面板附件可下载/打开 | 📋 计划中 |
| **回复/转发** | 邮件快速回复、全部回复、转发 | 📋 计划中 |
| **IMAP 状态指示器** | 连接状态 / 最后同步时间 / 手动刷新 | 📋 计划中 |
| **搜索增强** | 多条件搜索 (日期/附件/未读)、高亮 | 📋 计划中 |
| **邮箱设置向导** | 帮助用户自动检测 IMAP/SMTP 设置 | 📋 计划中 |
| **会话模式** | 邮件按主题分组，折叠展开 | 📋 计划中 |
| **全文搜索** | PostgreSQL tsvector 或 pgvector 语义搜索 | 📋 计划中 |
| **Sieve 过滤规则** | 服务端邮件过滤规则管理 | 📋 计划中 |
| **PGP 加密** | 端到端邮件加解密 | 📋 计划中 |
| **通知** | WebSocket / SSE 实时新邮件提醒 | 📋 计划中 |
| **响应式优化** | 触屏手势、邮件右滑操作 | 📋 计划中 |
| **AI Agent 集成** | MCP 协议、Function Calling、语义搜索、自动工作流 | 🔮 远期 |
| **插件系统** | Hook 事件机制、可扩展插件 | 🔮 远期 |
| **国际化** | 多语言、RTL 支持 | 🔮 远期 |
| **部署就绪** | Kubernetes、监控、备份 | 🔮 远期 |
