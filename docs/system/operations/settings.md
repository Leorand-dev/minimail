# ⚙️ 设置

## 邮箱账户

管理所有已配置的邮箱账户：

- **添加**: 填写 IMAP/SMTP 配置 → 测试连接 → 保存
- **删除**: 确认后移除账户
- **设为默认**: 写邮件时默认发件人
- 密码使用 Fernet 对称加密存储

## 个人设置

- 语言偏好（中文 / English）
- 主题（Light / Dark — 预留）
- 每页邮件数
- 预览面板开关

## 笔记库设置

在笔记库页面点击「⚙️ 设置」可管理：

- **公开分享开关**: 全局控制是否允许创建笔记分享链接
- **Webhook**: 添加/删除 Webhook URL，支持 note.created/updated/deleted 事件
- **快捷键**: 创建/删除侧栏快捷过滤入口

## 邮箱设置向导

在设置页面底部「📧 邮箱设置向导」：
1. 输入邮箱地址
2. 点击「自动检测」
3. 系统自动填充 IMAP/SMTP 配置
4. 点击「应用到设置」
