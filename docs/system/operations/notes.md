# 📝 笔记库

笔记库（Note）是基于 [Memos](https://github.com/usememos/memos) 架构解构自实现的轻量 Markdown 笔记系统，与邮件系统深度集成。

## 快速入口

- 侧栏「📝 笔记库」点击展开
- 子项：
  - **📋 所有笔记** — 时间线视图
  - **🏷️ 标签管理** — 创建/重命名/删除标签
  - **⚙️ 设置** — 笔记库偏好（预留）

## 创建笔记

1. 进入「📋 所有笔记」
2. 点击工具栏「✏️ 新建」
3. 在 TipTap 编辑器中编写内容（支持 Markdown：标题、粗体、斜体、列表、代码块、引用）
4. 在底部输入标签（逗号分隔，如 `meeting, project`）
5. 点击「创建」

## 查看笔记

笔记按创建时间倒序排列，每条笔记卡片显示：

- 📌 置顶标记
- 创建时间（相对时间：刚刚 / N 分钟前 / N 天前）
- Markdown 内容渲染
- `#标签`
- 已编辑标记

## 编辑笔记

- 悬停笔记卡片右上角显示 ✏️ 编辑按钮
- 点击进入编辑器，修改内容和标签
- 点击「保存」更新

## 置顶笔记

- 悬停笔记卡片点击 📌 置顶按钮
- 置顶笔记显示在列表顶部
- 再次点击取消置顶

## 删除笔记

- 悬停笔记卡片点击 🗑️ 删除按钮
- 软删除（`row_status = archived`），可通过 API 恢复

## 标签系统

### 自动提取
创建/编辑笔记时输入的标签自动提取，在笔记库左侧标签栏显示。

### 标签过滤
- 左侧标签栏点击标签名过滤笔记
- 点击「全部」清除过滤

### 标签管理
在「🏷️ 标签管理」面板：

- **创建**: 输入名称 → 点击「+ 创建」
- **重命名**: 悬停标签行 → 点击 ✏️ → 输入新名 → 确定
- **删除**: 悬停标签行 → 点击 🗑️ → 确认对话框
- 重命名时自动合并同名标签的笔记计数
- 删除标签时从所有笔记中自动移除

## 搜索笔记

在工具栏搜索框输入关键词：

- **全文搜索**: 基于 PostgreSQL tsvector
- **标签过滤**: 左侧标签栏可组合
- 结果按相关性排序

## AI Agent 集成

笔记库提供完整 REST API，AI Agent 可直接读写：

| Agent 场景 | API |
|-----------|-----|
| 记录决策 | `POST /api/notes` |
| 查询知识 | `GET /api/notes/search?q=` |
| 语义检索 | `POST /api/notes/search/semantic` (Phase 2) |
| 标签管理 | `GET /api/notes/tags` / `POST /api/notes/tags` / `PUT/DELETE` |

> 详情见 [AI Agent 操作](../operations/agent.md) 和 [笔记 API 参考](../api/notes.md)

## 分页

笔记列表使用游标分页（基于 `created_at`），每页最多 100 条，自动加载更多。
