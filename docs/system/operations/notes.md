# 📝 笔记库

笔记库（Note）是基于 [Memos](https://github.com/usememos/memos) 架构解构自实现的轻量 Markdown 笔记系统，与邮件系统深度集成。

## 快速入口

- 侧栏「📝 笔记库」点击展开
- 子项：
  - **📋 所有笔记** — 时间线视图
  - **🏷️ 标签管理** — 创建/重命名/删除标签
  - **⚙️ 设置** — 公开分享开关 / Webhook 管理 / 快捷键管理

## 创建笔记

1. 进入「📋 所有笔记」
2. 点击工具栏「✏️ 新建」
3. 在 TipTap 编辑器中编写内容（支持 Markdown：标题、粗体、斜体、列表、代码块、引用）
4. **自动标签**: 在内容中使用 `#tag` 语法，系统自动提取为标签
5. 可拖拽图片/文件到编辑器上传附件
6. 点击「创建」

## 查看笔记

笔记按创建时间倒序排列，每条笔记卡片显示：

- 📌 置顶标记
- 创建时间（相对时间：刚刚 / N 分钟前 / N 天前）
- Markdown 内容渲染
- `#标签`
- **内容属性**: 代码检测 / 链接检测 / 任务列表检测 图标
- **反应**: 👍❤️🎉 点击切换
- **评论**: 展开评论区查看/添加评论
- **附件**: 文件下载链接
- **分享**: 🔗 生成公开分享链接

## 编辑笔记

- 悬停笔记卡片右上角显示 ✏️ 编辑按钮
- 修改内容和标签，自动重新提取 `#tag`
- 支持置顶切换

## 搜索

- **全文搜索**: 在笔记库顶部搜索框输入关键词
- **语义搜索**: 由 AI Agent 调用 `POST /api/notes/search/semantic` 传入 embedding 搜索
- **统一搜索**: 🔍 全局搜索按钮跨邮件+笔记搜索

## 标签管理

- 「🏷️ 标签管理」子视图
- 创建新标签（重复名自动返回现有）
- 重命名标签（自动合并所有笔记中的标签名）
- 删除标签（从所有笔记中移除）

## 可见性

- **private**: 仅自己可见
- **protected**: 登录用户可见
- **public**: 任何人可见（配合分享链接）

## 分享

1. 在笔记卡片点击 🔗 图标
2. 系统生成分享链接
3. 将链接发送给任何人即可查看笔记（无需登录）
4. 可在「⚙️ 设置」→「公开分享」开关全局禁用

## Webhook

可在笔记库设置中添加 Webhook，支持事件：
- `note.created` — 笔记创建时触发
- `note.updated` — 笔记更新时触发
- `note.deleted` — 笔记删除时触发

## 快捷键

保存常用过滤条件为侧栏快捷入口：
1. 进入「⚙️ 设置」→「快捷键」
2. 填写名称和过滤标签
3. 保存后在侧栏「笔记库」分区下方显示
4. 点击快速切换到对应标签过滤
