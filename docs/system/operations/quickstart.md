# ⚡ 快速开始

## 登录

1. 访问 Minimail 前端页面
2. 输入邮箱/用户名和密码
3. 点击「登录」
4. 系统自动跳转到统一收件箱

> **测试账户**: `demo@example.com` / `Demo1234!`（演示用）

## 注册

1. 在登录页点击「注册」
2. 填写邮箱、用户名、密码
3. 提交后自动登录

## 初始设置

首次使用需要配置邮箱账户：

### 方式一：邮箱设置向导（推荐）

1. 点击左侧「⚙️ 设置」
2. 找到「📧 邮箱设置向导」
3. 输入邮箱地址，点击「自动检测」
4. 系统自动识别 IMAP/SMTP 服务器（支持 Gmail/QQ/163/Outlook 等 20+ 服务商）
5. 输入密码，点击「测试连接」
6. 保存

### 方式二：手动配置

1. 点击左侧「⚙️ 设置」→「邮箱账户」
2. 点击「+ 添加账户」
3. 填入 IMAP/SMTP 服务器地址、端口、密码
4. 点击「测试连接」验证
5. 保存

## 写邮件

1. 点击「✏️ 写邮件」
2. **发件人**: 如有多个邮箱账户，可下拉选择用哪个账户发送
3. 填写收件人（支持自动完成）
4. 使用 TipTap 富文本编辑器编写正文
5. 点击「发送」

## 笔记库

1. 点击侧栏「📝 笔记库」
2. 点击「✏️ 新建」创建笔记
3. 使用 `#标签` 语法自动添加标签
4. 支持 Reaction / 评论 / 附件上传
5. 点击 🔗 可生成公开分享链接

## 会话模式

在收件箱工具栏点击「💬 会话」，邮件按主题分组展示，点击展开查看组内消息。
