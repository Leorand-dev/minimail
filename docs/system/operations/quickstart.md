# ⚡ 快速开始

## 登录

1. 访问 Minimail 前端页面
2. 输入邮箱/用户名和密码
3. 点击「登录」
4. 系统自动跳转到统一收件箱

> **测试账户**: `demo@example.com` / `Demo1234!`（演示用）

## 注册

1. 在登录页点击「注册」
2. 填写邮箱、用户名、密码
3. 提交后自动登录

## 初始设置

首次使用需要配置邮箱账户：

1. 点击左侧「⚙️ 设置」→「邮箱账户」
2. 点击「+ 添加账户」
3. 填入：
   - 邮箱地址
   - IMAP 服务器地址（如 `imap.example.com`）
   - IMAP 端口（默认 993）
   - SMTP 服务器地址（如 `smtp.example.com`）
   - SMTP 端口（默认 465）
   - 邮箱密码
4. 点击「测试连接」验证设置
5. 保存账户
