# 📧 邮件系统

## 统一收件箱

所有已配置邮箱账户的收件箱汇聚为「统一收件箱」。

- 点击侧栏「📥 统一收件箱」查看所有账户的邮件
- 展开各邮箱账户可查看其独立文件夹树
- 未读数量以蓝色 badge 显示
- 点击邮件阅读后未读数自动更新

## 阅读邮件

1. 在邮件列表中点击任意邮件
2. 右侧预览面板显示邮件详情
3. 邮件自动标记为已读（文件夹未读数同步减少）
4. 支持附件预览和下载
5. 支持回复/全部回复/转发（自动预填收件人+引用原文）

## 写邮件

1. 点击侧栏「✏️ 写邮件」
2. **发件人**: 如有多个邮箱账户，可下拉选择用哪个账户发送
3. 填写收件人（支持自动完成，从通讯录匹配）
4. 支持 CC / BCC
5. 富文本编辑器（TipTap）：加粗、斜体、列表、引用
6. 可添加附件
7. 点击「发送」

## 搜索邮件

在工具栏搜索框中输入关键词，支持：
- 关键词高亮
- 日期范围筛选
- 仅未读筛选
- 按文件夹过滤

## 会话模式

点击工具栏「💬 会话」切换会话视图：
- 邮件按标准化主题分组（自动去除 Re:/Fwd:/回复/转发 前缀）
- 相同主题的邮件聚合显示
- 点击展开查看组内所有消息
- 支持按会话文件夹过滤

## 多账户管理

支持添加多个邮箱账户：
1. 进入「⚙️ 设置」→「邮箱账户」
2. 每个账户有独立的 IMAP/SMTP 配置
3. 可设置默认账户

## 邮箱设置向导

1. 进入「⚙️ 设置」→「📧 邮箱设置向导」
2. 输入邮箱地址
3. 点击「自动检测」
4. 系统匹配内置 20+ 服务商配置（Gmail/QQ/163/126/Outlook/iCloud/Yahoo/阿里云等）
5. 也可通过 MX 记录识别服务商
6. 点击「应用到设置」一键填充

## 快捷键

- `Ctrl+Enter` / `Cmd+Enter`: 发送邮件
- 侧栏文件夹点击切换
