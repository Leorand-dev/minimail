# 📧 邮件系统

## 统一收件箱

所有已配置邮箱账户的收件箱汇聚为「统一收件箱」。

- 点击侧栏「📥 统一收件箱」查看所有账户的邮件
- 展开各邮箱账户可查看其独立文件夹树
- 未读数量以蓝色 badge 显示

## 阅读邮件

1. 在邮件列表中点击任意邮件
2. 右侧预览面板显示邮件详情
3. 邮件自动标记为已读
4. 支持附件预览和下载

## 写邮件

1. 点击侧栏「✏️ 写邮件」
2. 填写收件人（支持自动完成，从通讯录匹配）
3. 支持 CC / BCC
4. 富文本编辑器（TipTap）：加粗、斜体、列表、引用
5. 可添加附件
6. 点击「发送」

**回复/转发**: 在预览面板点击「↩️ 回复」「↩️ 全部回复」「↪️ 转发」，自动预填收件人和引用原文。

## 搜索邮件

在工具栏搜索框中输入关键词，支持：

- 关键词高亮
- 日期范围筛选
- 仅未读筛选
- 按文件夹过滤

## 多账户管理

支持添加多个邮箱账户：

1. 进入「⚙️ 设置」→「邮箱账户」
2. 每个账户有独立的 IMAP/SMTP 配置
3. 可设为默认账户
4. 统一收件箱聚合所有账户邮件

## 文件夹操作

- 在侧栏邮箱账户下查看各文件夹
- 未读邮件计数自动更新
- 支持收件箱 / 已发送 / 草稿箱 / 垃圾邮件 / 已删除 / 归档

### 批量操作

邮件列表支持多选：

1. 点击邮件左侧 checkbox 选中
2. 表头 checkbox 全选当前页
3. 底部操作栏：「🗑 删除」「📂 移动到」
4. 移动到支持选择目标文件夹
5. 取消选择或点击「取消」按钮

## 已读状态

- 点击邮件后自动标记为已读
- 同一行显示粗体/正常字体区分未读/已读
- 切换视图后已读状态保持

## 实时连接状态

侧栏底部显示 IMAP 连接状态：

- 🟢 已连接：最后同步时间
- 🔴 未连接：可能无法收发邮件
- ↻ 刷新按钮：手动刷新连接状态

## 演示模式

未配置邮箱账户时自动加载演示数据（5 封示例邮件）：

1. 🎉 欢迎使用 Minimail
2. 📖 快速开始指南
3. 🔗 开源社区与贡献指南
4. 💡 AI Agent 集成小贴士
5. 🆕 更新日志
